Supporting Designers leveling up with the Skill Self-Evaluation

3 Weeks—2019: Leadership / Skill Development

Opportunity

The Designers at thoughtbot are generalists. They can work directly with stakeholders on design thinking workshops, then turn around and work in code. Because the role is broad, Designers struggled with identifying areas of growth, as well as classifying weaknesses.

Process

I wanted to create a tool that would allow Designers to see their progress in regards to skill growth. This would give them additional direction and the ability to more closely focus on skill development. Learn more about the process on the thoughtbot blog
